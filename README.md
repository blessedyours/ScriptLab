# ScriptLab

ScriptLab is a toolkit for Open.MP and SA:MP developers.

Generate Pawn commands faster, reduce repetitive work and speed up development.

## Features

* ZCMD Support
* Pawn.CMD Support
* Vanilla Pawn Support
* Dynamic Command Generation
* Weapon Selection
* Admin Level Configuration

## Getting Started

1. Download the repository.
2. Open index.html.
3. Configure your command.
4. Generate code.
5. Copy and use.

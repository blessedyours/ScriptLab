# ScriptLab

ScriptLab is a web-based tool created to simplify Pawn scripting for Open.MP and SA:MP developers.

The project currently focuses on generating common command templates, reducing the time spent writing repetitive code and helping developers prototype commands faster.

## Current Features (v1.0)

* ZCMD support
* Pawn.CMD support
* Vanilla Pawn support
* Basic command generation
* Simple and lightweight interface

## Why ScriptLab?

After repeatedly writing the same commands across different projects, I decided to build a tool that could generate the foundation automatically and speed up development.

ScriptLab is currently in its first public version and will continue to improve over time.
